## 🤝 Team 10 (212223) Ground Rules

### 📅 회의 규칙

- 매주 정기 미팅 최소 1회 진행
- 월 17:00-18:30, 목 15:30-18:30, 필요에 따라 온라인/오프라인 추가 편성 가능
- 안건은 회의 하루 전까지 공유하기
- 불참·지각 시 최소 2시간 전에 카톡으로 공지하기
- 지각하면 다음 회의 때 간식 사오기 🍩

### 💬 소통 규칙

- 주요 소통은 카카오톡 단톡방 사용
- 비대면 소통 (작업물 관리 등) 디스코드 사용
- 메시지 확인 후 24시간 이내 답장하기
- 모르는 건 솔직하게 물어보기 (눈치 보지 않기)

### 🎯 역할 규칙

- 각 팀원은 핵심 역할 1개 이상 담당
- 고윤진: 제출 담당
- 김나현: 최종 merge, 깃허브 관련 제출 담당
- 이유진 : 노션 기록및 프로젝트 진행상황 관리
- 역할 변경이 필요하면 팀 합의 후 진행
- 결과보다 과정에서의 노력을 서로 인정해주기

### 💻 작업 규칙

- GitHub로 코드 관리, PR 머지 전 팀원 1명 이상 리뷰 필수
- 작업하며 계속 상황 공유하기
- 작업 중 막히는 부분은 혼자 끙끙대지 말고 팀원한테 바로 공유하기

### 🔥 갈등 해결 규칙

- 문제 발생 시 단톡방보다 팀 회의에서 직접 논의하기
- 해결이 어려우면 지도교수님께 상담 요청 (심재형 교수님 / jh.sim@ewha./ac.kr)
- 어떤 의견이든 일단 끝까지 들어주기
