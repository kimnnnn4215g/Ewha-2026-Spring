#Team10 Ground Rule Dummy File

1. 회의 규칙
매주 최소 1회 팀 미팅을 진행한다.
지각 또는 불참 시 미리 팀 채팅방에 공지한다.
회의 후 회의 결과와 할 일을 정리하여 공유한다.
2. 커뮤니케이션 규칙
주요 소통은 Slack 또는 카카오톡을 사용한다.
메시지에 대한 응답은 24시간 이내에 한다.
의견을 제시할 때는 상대방을 존중하는 태도로 소통한다.
3. 역할 규칙
각 팀원은 최소 1개의 핵심 역할을 담당한다.
역할 변경이 필요할 경우 팀원 전체의 합의 후 변경한다.
맡은 역할에 대한 책임을 가지고 작업을 수행한다.
4. 작업 규칙
GitHub를 통해 코드 및 자료를 관리한다.
주요 작업은 Pull Request를 통해 공유 및 검토한다.
작업 진행 상황은 정기적으로 팀원에게 공유한다.
5. 일정 관리 규칙
모든 작업은 정해진 마감 기한을 준수한다.
일정 변경이 필요한 경우 사전에 팀원에게 공유한다.
6. 갈등 해결 규칙
문제가 발생하면 팀 내에서 먼저 논의하여 해결한다.
개인 비난이 아닌 문제 해결 중심으로 의견을 제시한다.
